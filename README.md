bili-cli
=========

哔哩哔哩视频命令行下载器

## 实现功能

- [x] 用户
  - [x] 登录
  - [x] 个人信息
- [x] 视频下载
  - [x] 高清视频下载并合并
  - [x] BV下载
  - [x] EP/SS下载
  - [x] 来自手机的短视频/短链接
  - [ ] 断点续传
  - [ ] 下载过滤器

## 如何使用

1. 将ffmpeg命令添加到PATH (合并必须使用)
2. 运行本软件
3. task.db是下载进度数据, (计划中)未完结EP下载完成后不删除, 以便以后继续下载新的章节。
